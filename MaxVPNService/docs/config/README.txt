These are simple example WireGuard configuration files for both the client and server, designed to work together.

1. Run the server configuration on a server that has internet access.

2. Take the server's public IP address and enter it into the Endpoint field of the client's configuration file.

3.Then, run the client configuration on your client device to establish the connection.

make sure you open port 51820 on the server
